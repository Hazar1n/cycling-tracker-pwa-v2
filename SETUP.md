# Cycling Tracker Pro — Setup Guide

## 1. Supabase — добави нови колони

Влез в [supabase.com](https://supabase.com) → твоя проект → **Table Editor** → таблица `workouts`.

Добави следните колони (ако ги няма):

| Колона  | Тип      | Default |
|---------|----------|---------|
| `km`    | `float8` | `0`     |
| `notes` | `text`   | `null`  |

**Как:** Table Editor → `workouts` → "+ Add column" за всяка.

---

## 2. Файлова структура

```
/твоя-папка/
├── index.html          ← главният файл
└── images/
    └── cycle.jpg       ← фоново изображение (по желание)
```

Ако нямаш `images/cycle.jpg`, махни тези редове от CSS-а в `index.html`:

```css
url('images/cycle.jpg') center/cover no-repeat;
```

---

## 3. Стартиране

**Локално (за тест):**
```bash
npx serve .
# или
python3 -m http.server 8080
```
Отвори `http://localhost:8080`

**Хостване (безплатно):**
- [Netlify Drop](https://app.netlify.com/drop) — провлачи папката
- [Vercel](https://vercel.com) — `vercel deploy`
- GitHub Pages

## Нови функции

| Функция | Описание |
|---------|----------|
| 🗂️ **Таб навигация** | Днес / Седмица / Добави / Статистика |
| 🔄 **Анимиран прогрес ринг** | Плавна анимация вместо права лента |
| ⏱️ **Таймер** | Старт/Пауза/Нулиране на тренировката |
| 🔥 **Streak** | Брой поредни дни с тренировка |
| 🛣️ **Километри** | Записване и обобщение на изминати км |
| 📝 **Бележки** | Текстова бележка към всяка тренировка |
| 📊 **3 графики** | Тегло / Калории / Км |
| 📅 **Седмично сравнение** | Тази vs миналата седмица |
| 🎉 **Confetti** | При завършена седмица вместо alert() |
| 🗑️ **Изтриване** | Изтриване на записи от историята |
| 💬 **Toast съобщения** | Без alert() — стилни нотификации |
